# Employee Data Analysis System (C++)

## 📌 Overview
This project is a C++ program that reads employee data from a file and performs basic data analysis.

It demonstrates the use of:
- File handling
- Structures
- Functions
- Data processing

---

## 🚀 Features
- Read employee data from file (`data.txt`)
- Store data using structures
- Calculate average salary
- Find employee with highest salary
- Display results in console

---

## 📂 File Format

Create a file named `data.txt`:
John 101 5000 25
Alice 102 7000 30
Bob 103 4500 28

Format:
Name ID Salary Age

---

## 📤 Output Example

Average Salary: 5500
Highest Paid Employee:
Name: Alice
Age: 30

---

## 💡 Concepts Used
- Structures (struct)
- File Input (ifstream)
- Vectors
- Functions
